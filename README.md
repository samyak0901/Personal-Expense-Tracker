# Personal-Expense-Tracker

## Description
A simple command-line tool to help users track their daily spending. It allows users to input items, costs, and categories, then saves them to a CSV file for long-term tracking.

## How to Set Up
1. Ensure you have **Python 3.x** installed.
2. Download `main.py` from this repository.

## How to Use
1. Run the script using the command: `python main.py`.
2. Select '1' to add an expense.
3. Select '2' to see everything you've spent.
4. Select '3' to see your grand total.

## Features
- Data persistence (saves your info even if you close the app).
- Simple category sorting.
- Automatic total calculation.
